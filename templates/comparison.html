Protocol Telemetrics

A comparative analysis of priority inversion handling across the scheduling algorithms within the control plane.

Protocol Execution Time Delay Impact Mechanism Description
No Protocol High (~5.0s) Unbounded Delay Standard preemption allows medium-priority tasks to infinitely block high-priority tasks waiting on resources held by low-priority tasks.
Priority Inheritance (PIP) Medium (~3.0s) Bounded Delay Resource-holding tasks temporarily inherit the priority of blocked tasks. Prevents medium-priority task preemption, reducing delay.
Priority Ceiling (PCP) Low (~1.5s) No Delay (Optimal) Each resource has a ceiling priority. Tasks dynamically execute at the ceiling priority when holding a resource, preventing deadlocks and transitive blocking entirely.
